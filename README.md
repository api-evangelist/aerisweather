# AerisWeather

Comprehensive weather data REST API providing forecasts, observations, severe weather alerts, historical data, satellite imagery, and maps for global locations. Now operating as Xweather (acquired by Vaisala in 2022), the platform offers 60+ endpoints covering current conditions, forecasts, air quality, lightning, maritime weather, road conditions, and more.

- **Website:** https://www.xweather.com
- **Documentation:** https://www.xweather.com/docs/weather-api
- **GitHub:** https://github.com/aerisweather
- **Status:** https://xweatherstatus.com/
- **Blog:** https://www.xweather.com/blog
